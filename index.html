HTML Tables

HTML Tags
Tag Name Description
<table> Table The main table tag
<thead> Table Head Column titles
<caption> Table Caption Tiltle of the Table
colgroup HTML Table Colgroup Should be used as a container for column specifications. Each group is specified with a <col> element and a span attribute.
<tbody> Table Body The main part of the table
<tr> Table Row Defines the table row
<td> Table cell The content of the cell
<tfoot> Table Footer Summing up numberic values from each row
CSS Attributes
Attribute Name Description
border-collapse Collapse Table Border Sets whether the table borders should be collapsed into a single border
colspan Colspan Merging the column
rowspan Rowspan Merging the rows
vertical-alignment Vertical Alignment property sets the vertical alignment (like top, bottom, or middle) of the content in <th> or <td>
